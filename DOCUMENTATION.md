

## Features

- No significant modifications to existing features, only added a new function

### traverseAndPrint
Traverses a linked list and prints its elements.

- **Input**: head: The head node of the linked list.
- **Output**: None, prints the linked list elements.

### Node
Represents a node in a linked list.

- **Input**: data: The data to be stored in the node.
- **Output**: A new Node object.

### newfunc1
Removes all space characters (' ') from a string and replaces them with '1'.

- **Input**: input_string
- **Output**: Modified string with spaces replaced by '1'

### trim_all_spaces
Removes all space characters from a string and replaces them with hyphens

- **Input**: Input string
- **Output**: String with all spaces replaced with hyphens
