### inorder_traversal
Performs in-order traversal of a binary tree and returns a list of node values.

- **Input**: root node of the binary tree
- **Output**: list of node values in in-order traversal order

### preorder_traversal
Performs pre-order traversal of a binary tree and returns a list of node values.

- **Input**: root node of the binary tree
- **Output**: list of node values in pre-order traversal order

### TreeNode
Represents a node in a binary tree with a value and optional left and right child nodes.

- **Input**: value, left child node, right child node
- **Output**: TreeNode object

### newfunc1
Removes all space characters (' ') from a string and replaces them with '1'.

- **Input**: input_string
- **Output**: Modified string with spaces replaced by '1'

### trim_all_spaces
Removes all space characters from a string and replaces them with hyphens

- **Input**: Input string
- **Output**: String with all spaces replaced with hyphens
